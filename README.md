## 🌟 О проекте

**«ПРОЙДУ?»** — это федеральный навигатор, который помогает абитуриентам и их родителям точно оценить шансы на поступление. В отличие от стандартных калькуляторов, мы учитываем **индивидуальные баллы по каждому предмету**, минимальные пороги, альтернативные группы экзаменов и реальные исторические срезы.

> 💡 **Честность превыше всего:** Интерфейс прозрачно показывает, какие программы прошли верификацию по официальным источникам, а какие находятся в процессе наполнения. Мы не обещаем охватить все вузы страны за один день, но гарантируем точность там, где данные уже загружены.

## 📊 Текущий охват данных (Live Data)

На данный момент система масштабируется до федерального уровня и включает:
- 🗺️ **89 регионов** РФ.
- 🏛️ **751 вуз и филиал** в национальном каталоге.
- 🎯 **9 верифицированных программ 2026 года** в слое конкурентного поиска.
- 🧪 **Успешный тест мульти-вузовского маршрута:** ВШЭ, МФТИ, КФУ и ИТМО.

*Код полностью готов к федеральному масштабированию (ingestion), но база данных еще наполняется документами приема всех российских университетов.*

## ✨ Ключевые возможности

### 🔍 Умный поиск
- Поиск по **индивидуальным баллам ЕГЭ** (а не только по общей сумме).
- Автоматическая проверка обязательных предметов, пороговых баллов и альтернативных групп экзаменов.
- Фильтрация по вузу, названию программы, коду специальности, региону и городу.

### 🗺️ Режимы построения маршрута
- **Route candidates:** Построение индивидуальной траектории поступления.
- **Reachable:** Программы, доступные по историческим срезам прошлых лет.
- **Compatible:** Все совместимые программы по набору экзаменов.
- Поддержка результатов для **нескольких вузов и городов** одновременно.

### 📋 Полная траектория абитуриента (Developer Full Route)
- Учет индивидуальных достижений (портфолио).
- Диверсификация приоритетов при подаче заявлений.
- Отслеживание дедлайнов и чек-листы необходимых документов.
- Честные лейблы покрытия и ссылки на первоисточники.

## 🛠 Технологии и стек

- **Frontend:** React / Vite (оптимизировано для GitHub Pages и Telegram Mini App).
- **Backend & DB:** Supabase (PostgreSQL, Edge Functions, RLS).
- **CI/CD & Hosting:** GitHub Actions, GitHub Pages.
- **Интеграции:** Telegram Bot API, Telegram Stars (в разработке).
